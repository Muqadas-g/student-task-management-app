## 📌 Project Overview

A full-stack **Student Task Management Application** that allows users to manage projects and tasks efficiently.

Users can:

* Sign up & log in securely 🔐
* Create and manage projects 📁
* Track task progress ✅
* Store data in a cloud database ☁️

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Muqadas-g/student-task-management-app.git
```

### 2️⃣ Go to Project Folder

```bash
cd student-task-management-app
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Configure Environment Variables

Create a `.env` file and add:

```
VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
```

### 5️⃣ Run Project

```bash
npm run dev
```

## 🔐 Security

* Authentication handled by **Supabase Auth**
* **Row Level Security (RLS)** ensures users only access their own data

---

## 🗄️ Database Structure

* **users** → stores user information
* **projects** → stores project details
* **tasks** → stores tasks linked with projects
